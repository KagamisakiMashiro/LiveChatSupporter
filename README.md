# LiChaS(リチャス) - Live Chat Supporter

本ソフトウェアはYouTube配信時に、コメントを処理して配信者をサポートするために開発されています。  
現在はアンケート機能と特定の視聴者からのコメントを表示する機能が搭載されています。

## 機能
アンケート機能は視聴者がコメント欄にアルファベット（半角・全角・大文字・小文字も対応）すれば、投票できます。  
最大26個の選択肢も対応しています（画面に表示できるかどうかは別ですが…）。  
アンケート表示していない時、画面がただのGBなので、ずっと表示しつづいても大丈夫です。  

コメント機能は、特定の視聴者からのコメントを長く表示する機能です。  
指定方式はチャンネルIDの指定、ユーザー名の指定とモデレーター（通称スパナ）を対応しています。  
表示時間、表示コメント数ももちろん設定できます。  
配置方式は列数で設定できます。（0にしたら、全部同じ行に表示します。）

## 使用条件
一言でいうと、特にありません。  
このソフトウェアは個人利用・商用利用など全部許可しております。  
Apacheライセンス2.0を使用していますので、ソースの流用・改造なども自由です。
